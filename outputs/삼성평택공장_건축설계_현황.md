## 1. 평택공장 건축 프로젝트 개요

### 프로젝트 기본 정보

삼성전자의 평택캠퍼스는 반도체 생산의 핵심 거점으로, 현재 다음과 같은 프로젝트들이 진행 중입니다:

| 항목 | P4(4공장) | P5(5공장) | P6(6공장) |
|------|---------|---------|---------|
| **프로젝트명** | 평택 4공장 건설 | 평택 5공장(AI HBM팹) | 평택 6공장 |
| **프로젝트 규모** | 3층 건물 구조 | 가로 650m, 세로 195m, 3층 | 계획 중 |
| **전체 투자규모** | 약 3조7,667억원 | 약 60조원 이상 | 계획 중 |
| **예정 착공** | 2021년(이미 시작) | 2025년 1월 이후 | 추가 계획 중 |
| **목표 준공/가동** | 2027년 4월 | 2028년 3월 | 미정 |

### 평택 P5(5공장) - 최신 프로젝트

#### 무엇인가요?
삼성전자가 AI 시대를 대비해 지으려는 **새로운 메가 반도체 공장**입니다.

**쉽게 말하면**,
- 인공지능(AI) 칩의 핵심 부품인 '고대역폭메모리(HBM)'를 만드는 초현대식 공장
- 프로그래밍할 수 있는 칩(파운드리)도 한 곳에서 생산
- 기존 공장의 1.5배 규모로 건설

#### 투자 규모
- **60조원 이상** (기존 공장의 2배 이상 투자)
- 한국의 대규모 프로젝트 중 최대 규모

#### 건물 규모
- 가로 650미터 × 세로 195미터
- 총 3층 규모 (당초 4층에서 3층으로 조정)
- 클린룸 6개 탑재 (반도체 생산에 필수적인 초청정 구역)

## 3. 건축 설계 단계별 현황

### **P4(4공장) 진행 상황**

#### 프로젝트 개요
- **착공 시점**: 2021년
- **규모**: 원래 파운드리(반도체 대량 생산) 중심으로 설계
- **최근 변화**: D램(메모리 칩)과 HBM(고대역폭메모리) 중심으로 재편

#### 설계 단계별 진행 상황

| 단계 | 상태 | 진행도 | 설명 |
|------|------|--------|------|
| **기본 설계(Basic Design)** | ✓ 완료 | 100% | 2021년 설계 완료 후 변경 설계 진행 중 |
| **상세 설계(Detailed Design)** | ✓ 완료 | 100% | 설계 단계 완료 |
| **건축 허가/인허가** | ✓ 완료 | 100% | 인허가 완료 |
| **1~3구역 건축공사** | ✓ 완료 | 90~100% | Ph1, Ph3 거의 마무리 단계 |
| **4구역 건축공사** | 진행 중 | 70% | 2025년 4월 마무리 예정 |
| **배관·전기 공사** | 준비 중 | 10% | 4구역 건축 후 본격화 예정 |
| **장비 설치** | 진행 중 | 5% | 일부 구역에서 장비 설치 시작 |
| **완공** | 예정 | - | 2027년 4월 목표 |

#### 세부 진행 사항
- **Ph1, Ph3**: 공사 사실상 완료 단계 진입
- **Ph2, Ph4**: 내부 공사 본격 준비 중
- **2025년 예정**: 1~3구역 마무리, 4구역 착공 가속
- **배관/전기 공사**: 2025년 4월 이후 본격화
- **장비 반입**: 일부 라인부터 시작 (HBM4 우선)

#### 예정 완공 시점
- **2027년 4월**: 전체 건축공사 완료
- **2027년 상반기**: 장비 설치 및 테스트
- **2027년 하반기~**: 본격 가동

| 단계 | 상태 | 진행도 | 설명 |
|------|------|--------|------|
| **기본 설계(Basic Design)** | ✓ 완료 | 100% | 2024년 완료 (4층에서 3층으로 수정) |

#### 세부 진행 사항
- **2024년**: 설계 수정 완료 (4층 → 3층으로 축소)
  - 당초 4층 설계에서 3층으로 변경
  - 클린룸 8개 → 6개로 축소
  - 이유: AI칩 긴급 생산 필요로 인한 신속 착공 결정

- **2025년 상반기**: 사전 공사 진행
  - 부지 정지 및 기초 작업
  - 관리자 일부 복귀 시작

- **2025년 4월**: 골조 공사 착수 예정
  - 콘크리트 양생 시간(약 1개월) 고려하여 4월 1일 착공 계획
  - 골조 세우기 작업 개시
  - 본격 인력 투입 시작 (1만 5천~2만 명)

- **2025년 하반기~2026년 상반기**: 상부 구조물 공사
  - 상층부 건설 진행

- **2027년**: 클린룸 마감공사 시작
  - Ph1(첫 번째 클린룸) 기준으로 2027년부터 마감공사 시작

- **2027년 하반기~2028년**: 장비 설치 및 테스트

- **2028년 3월**: 최초 가동 목표

#### 핵심 특징
- **통합 메가 팹(Mega Fab)**: HBM + 파운드리 + 패키징을 한 동선으로 통합
- **AI 중심 설계**: HBM4E, HBM5 등 AI칩용 고성능 메모리 생산
- **3층 구조**: 층당 2개 클린룸으로 효율적 배치

## 6. 용어 설명 (비전공자를 위한 가이드)

### **반도체 공장(FAB) 관련 용어**

| 용어 | 의미 | 쉬운 설명 |
|------|------|---------|
| **FAB** | Fabrication Plant | 반도체를 실제로 만드는 공장 |
| **HBM** | High Bandwidth Memory | AI 칩이 빠르게 생각하도록 도와주는 초고속 메모리 |
| **파운드리** | Foundry | 다른 회사의 설계를 받아서 칩을 대량으로 만들어주는 대행 공장 |
| **클린룸** | Clean Room | 먼지 하나도 없어야 하는 초청정 구역 (반도체는 미세하므로 필요) |
| **EUV** | Extreme Ultra Violet | 극도로 미세한 회로를 그리는 초첨단 기술 |
| **D램** | Dynamic RAM | 컴퓨터의 임시 저장 메모리 |
| **NAND** | NAND Flash | USB, SSD 등에 들어가는 저장 메모리 |

### **건축 공사 단계 관련 용어**

| 용어 | 의미 | 순서 |
|------|------|------|
| **기본 설계** | 프로젝트의 전체적인 모양과 구조 설계 | 1번 |
| **상세 설계** | 기본 설계를 바탕으로 세부 부분 설계 | 2번 |
| **건축 허가** | 정부가 '이 설계대로 지어도 좋다'는 승인 | 3번 |
| **골조 공사** | 건물의 뼈대(기둥, 보, 바닥) 만드는 공사 | 4번 |
| **마감 공사** | 벽, 바닥, 천장, 도장 등 마무리하는 공사 | 5번 |
| **배관·전기** | 물, 가스, 전기 등 설비 설치 | 6번 |
| **장비 설치** | 실제 생산에 필요한 기계 설치 | 7번 |
