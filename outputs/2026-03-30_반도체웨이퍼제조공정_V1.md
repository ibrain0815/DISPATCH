# 반도체 웨이퍼 제조 공정과 화학약품

## 🔍 웨이퍼란?

**웨이퍼** = 반도체 칩을 만드는 기초 재료
- 실리콘(Si)으로 만든 얇은 원판
- 지름: 150mm, 200mm, 300mm (요즘은 300mm 주류)
- 두께: 약 0.75mm (매우 얇음)
- 이 위에 수천 개의 트랜지스터를 새겨넣음

---

## 🏭 반도체 웨이퍼 제조 공정 전체 흐름

```
1️⃣ 웨이퍼 준비
   ↓
2️⃣ 산화 (Oxidation)
   ↓
3️⃣ 포토리소그래피 (Photolithography)
   ↓
4️⃣ 식각 (Etching)
   ↓
5️⃣ 증착 (Deposition)
   ↓
6️⃣ 도핑 (Doping)
   ↓
7️⃣ 폴리싱 (Polishing)
   ↓
8️⃣ 세정 (Cleaning)
   ↓
9️⃣ 검사 (Inspection)
   ↓
🔟 반복 (각 제조 단계를 여러 번 반복)
```

---

## 📊 주요 공정별 상세 설명

### 1️⃣ 웨이퍼 준비 (Wafer Preparation)

#### 목적
순수한 실리콘 웨이퍼를 다음 공정에 사용 가능하도록 준비

#### 과정
```
실리콘 잉곳 생성 (실리콘 녹였다가 식혀서 만든 큰 막대)
   ↓
슬라이싱 (다이아몬드 톱으로 매우 얇게 자르기)
   ↓
연마 (표면을 매끄럽게 만들기)
   ↓
세정 (먼지와 오염물 제거)
   ↓
순수한 웨이퍼 완성
```

#### 사용 화학약품
- **DI수** (증류수): 표준 세정
- **H2SO4** (황산): 유기물 제거
- **HF** (불산): 자연 산화층 제거
- **HNO3** (질산): 산화 작용

---

### 2️⃣ 산화 (Oxidation)

#### 목적
**실리콘 표면에 절연층(SiO2) 만들기**
- 마치 종이 위에 투명한 플라스틱 필름을 입히는 것처럼
- 이 층이 회로의 절연 역할

#### 과정
```
웨이퍼를 오븐처럼 뜨거운 산화로에 넣음 (약 800~1,200°C)
   ↓
산소 또는 수증기를 주입
   ↓
Si + O2 → SiO2 (실리콘 표면이 산화됨)
   ↓
약 1,000nm 정도의 얇은 SiO2 층 형성
```

#### 화학 반응
```
건식 산화 (Dry Oxidation):
Si + O2 → SiO2

습식 산화 (Wet Oxidation):
Si + H2O → SiO2 + H2↑
```

#### 특징
- ✅ 속도가 빠름 (습식 > 건식)
- ✅ 품질 우수 (건식 > 습식)
- ✅ 온도 높을수록 빠름

---

### 3️⃣ 포토리소그래피 (Photolithography)

#### 목적
**복잡한 회로 패턴을 웨이퍼 위에 전사하기**
- 마치 사진을 종이에 인쇄하는 것처럼
- 나노 수준의 초정밀 패턴 (7nm, 5nm 등)

#### 과정 (4단계)

**Step 1: 포토레지스트 코팅**
```
웨이퍼 전체에 포토레지스트(감광제) 층을 얇게 도포
- 액체 상태의 감광성 물질
- 스핀 코팅으로 균일하게 펴짐
```

**Step 2: 노광 (Exposure)**
```
마스크(판화 판처럼 투명/불투명 부분이 있음) 통해 UV 광선 조사
   ↓
빛이 닿은 부분 (또는 닿지 않은 부분)의 포토레지스트가 화학 변화
   ↓
노광 후 웨이퍼 패턴 형성 시작
```

**Step 3: 현상 (Development)**
```
특수 용액(현상액)에 담그기
   ↓
화학 반응으로 용해된 포토레지스트 제거
   ↓
패턴이 명확하게 드러남
```

**Step 4: 검사 및 경화**
```
패턴이 올바른지 확인
   ↓
베이크(가열)로 포토레지스트 경화 및 강화
```

#### 사용 화학약품

| 약품 | 역할 | 예시 |
|------|------|------|
| **포토레지스트** | 감광 물질 | 노바렉 (Novatech), JSR |
| **현상액** (Developer) | 용해 제거 | TMAH (4-25% aqueous solution) |
| **린스액** | 헹굼 | DI수, IPA (이소프로필알콜) |
| **스트립액** | 제거 | H2SO4/H2O2 혼합액 |

#### 특징
- ⚠️ 가장 복잡한 공정
- ⚠️ 미세한 먼지 하나도 불량 원인
- ⚠️ 클린룸 필수 (Class 1~10 수준)

---

### 4️⃣ 식각 (Etching)

#### 목적
**포토레지스트 패턴에 따라 아래 층(SiO2 또는 Si)을 선택적으로 제거**
- 마치 조각칼로 원하는 부분만 깎아내는 것처럼

#### 두 가지 방식

**방식 1: 드라이 에칭 (Dry Etching) - 플라즈마 에칭**
```
특수 가스(CF4, Cl2 등)를 플라즈마 상태로 만듦
   ↓
플라즈마가 노출된 표면을 화학/물리적으로 제거
   ↓
매우 정확하고 미세한 패턴 가능
   ↓
반도체는 주로 이 방식 사용
```

**방식 2: 웨트 에칭 (Wet Etching) - 화학 용액**
```
특수 화학 용액에 담그기
   ↓
화학 반응으로 노출된 표면 용해
   ↓
정확도는 낮지만 빠르고 저비용
   ↓
구형 디바이스에 주로 사용
```

#### 사용 화학약품

| 에칭 대상 | 에칭액 | 반응식 |
|----------|--------|--------|
| **SiO2** (산화 실리콘) | HF (불산) | SiO2 + 6HF → H2SiF6↑ + 2H2O |
| **Si** (실리콘) | HF + HNO3 혼합액 | 3Si + 4HNO3 + 18HF → 3H2SiF6 + 4NO↑ + 8H2O |
| **금속** (알루미늄 등) | 다양한 산 혼합액 | - |

#### 특징
- ⚠️ 불산(HF)은 극도로 위험한 화학물질
- ⚠️ 환기 필수, 누수 위험 높음
- ✅ 정확도 결정하는 중요한 공정

---

### 5️⃣ 증착 (Deposition)

#### 목적
**새로운 물질 층을 웨이퍼 위에 입히기**
- 금속(금, 구리, 알루미늄), 반도체(폴리실리콘), 절연물(SiO2) 등
- 마치 벽화를 그릴 때 밑바탕을 칠하는 것처럼

#### 주요 증착 방식

**1. CVD (Chemical Vapor Deposition) - 화학 기상 증착**
```
기체 상태의 원료를 뜨거운 웨이퍼에 공급
   ↓
화학 반응으로 고체 물질이 웨이퍼 표면에 형성
   ↓
균일하고 정확한 두께 제어 가능
```

**2. PVD (Physical Vapor Deposition) - 물리 기상 증착**
```
고에너지로 물질을 기화시킴 (스퍼터링)
   ↓
웨이퍼 표면에 침착
   ↓
금속 층 증착에 주로 사용
```

**3. ALD (Atomic Layer Deposition) - 원자층 증착**
```
화학물질을 한 층씩 매우 정확하게 쌓음
   ↓
극도로 얇고 균일한 층 형성 가능 (1nm 단위)
   ↓
최신 미세공정의 필수 기술
```

#### 사용 화학약품

| 증착 물질 | 원료 가스 | 용도 |
|----------|----------|------|
| **SiO2** (이산화규소) | SiH4 (실란) + O2 | 절연층 |
| **Si3N4** (질화 실리콘) | SiH4 + NH3 | 절연층, 배리어 |
| **폴리실리콘** | SiH4 | 게이트 전극 |
| **Al** (알루미늄) | AlCl3 증기 | 배선 |
| **Cu** (구리) | Cu(II) 화합물 | 배선 (현재 주류) |
| **W** (텅스텐) | WF6 (육불화텅스텐) | 컨택 플러그 |

#### 특징
- ⚠️ 독성 가스 사용 (SiH4, WF6 등)
- ✅ 균일한 두께 제어 가능
- ✅ 미세공정의 핵심 공정

---

### 6️⃣ 도핑 (Doping)

#### 목적
**반도체에 불순물을 주입하여 전기 특성 변화**
- 순수한 실리콘은 반도체가 아님
- 불순물(도판트)을 넣어야 전기를 전도
- n형(음극) 또는 p형(양극) 만들기

#### 두 가지 방식

**방식 1: 이온 주입 (Ion Implantation)**
```
도판트 이온을 고에너지로 가속
   ↓
웨이퍼 표면에 고속 충돌
   ↓
이온이 실리콘 내부에 매장됨
   ↓
깊이와 농도 정밀 제어 가능
```

**방식 2: 확산 (Diffusion)**
```
고온에서 도판트 가스 공급
   ↓
열의 영향으로 도판트가 실리콘 내부로 퍼짐
   ↓
비용 저렴하지만 정밀도는 낮음
```

#### 사용 화학약품 (도판트)

| 도판트 | 특성 | 용도 |
|--------|------|------|
| **P** (인) | 5가 원소 (전자 공급) | n형 반도체 |
| **As** (비소) | 5가 원소 | n형 반도체 (깊은 접합) |
| **B** (붕소) | 3가 원소 (정공 공급) | p형 반도체 |
| **Sb** (안티모니) | 5가 원소 | n형 반도체 (깊은 접합) |

#### 원리
```
순수 Si (4가)
   + P (5가)
   = Si-P 결합에서 여분의 전자 생성
   → n형 반도체 (전자 전도)

순수 Si (4가)
   + B (3가)
   = Si-B 결합에서 정공(구멍) 생성
   → p형 반도체 (정공 전도)
```

#### 특징
- ⚠️ 고온 공정 필요
- ✅ 트랜지스터의 전기 특성 결정
- ✅ 정밀한 임플란트 깊이 제어

---

### 7️⃣ 폴리싱 (Polishing) - CMP

#### 목적
**웨이퍼 표면을 평탄하게 만들기**
- 앞의 공정들로 울퉁불퉁해진 표면 정리
- Chemical Mechanical Polishing (CMP)
- 화학 반응 + 기계적 연마 조합

#### 과정
```
웨이퍼를 회전판(패드)에 올림
   ↓
연마재(슬러리) 공급 (미세한 입자 + 화학액)
   ↓
화학 반응으로 물질 부분적으로 약화
   ↓
기계적 마찰로 제거
   ↓
초평탄 표면 달성
```

#### 사용 화학약품 (슬러리 구성)

| 성분 | 역할 | 예시 |
|------|------|------|
| **연마재** | 기계적 제거 | 실리카 입자 (SiO2), 산화세륨 (CeO2) |
| **산화제** | 화학적 약화 | H2O2 (과산화수소) |
| **산/염기** | pH 조절 | H2SO4, KOH |
| **계면활성제** | 입자 분산 | KI, KIO3 혼합액 |

#### 특징
- ✅ 극도로 평탄한 표면 (RMS < 1nm)
- ⚠️ 매우 정밀한 공정
- ⚠️ 과도한 연마 금지 (층 손상)

---

### 8️⃣ 세정 (Cleaning)

#### 목적
**이전 공정에서 남은 오염물질 완전 제거**
- 화학약품 잔여물, 금속 이온, 유기물 등
- 매우 순수한 상태 필요 (1ppt 수준)

#### 세정 방식

**표준 RCA 세정 (반도체 업계 표준)**

```
Step 1: SC-1 세정 (수산화암모늄 + 과산화수소)
   ↓
NH4OH + H2O2 혼합액
   ↓
유기물과 일부 금속 제거
   ↓
온도: 70~80°C, 시간: 10~15분

Step 2: 헹굼 (DI수)
   ↓
열화학적으로 2~3회 반복

Step 3: SC-2 세정 (염산 + 과산화수소)
   ↓
HCl + H2O2 혼합액
   ↓
금속 이온 제거 (특히 Fe, Cu, Zn 등)
   ↓
온도: 70~80°C, 시간: 10~15분

Step 4: 최종 헹굼 및 건조
   ↓
DI수로 여러 번 헹굼
   ↓
스핀 드라이로 건조
```

#### 사용 화학약품

| 약품 | 성분 | 용도 |
|------|------|------|
| **SC-1** | NH4OH + H2O2 | 유기물 제거 |
| **SC-2** | HCl + H2O2 | 금속 이온 제거 |
| **DI수** | H2O (초고순도) | 헹굼 |
| **IPA** | 이소프로필알콜 | 건조 촉진 |

#### 특징
- ✅ 다회 반복으로 고순도 달성
- ⚠️ 화학약품 혼합 시 발열 위험 (안전!)
- ⚠️ 환경 영향 큼 (폐액 처리 중요)

---

### 9️⃣ 검사 (Inspection & Testing)

#### 목적
**각 공정 단계에서 품질 확인**
- 결함 조기 발견
- 불량품 배제

#### 검사 방법

**비파괴 검사**
```
1. 광학 검사 (Optical Inspection)
   - 현미경으로 패턴, 결함 확인
   - SEM (주사 전자 현미경)으로 미세 구조 관찰

2. 전기적 검사 (Electrical Testing)
   - 저항, 용량성 측정
   - 누설 전류 확인

3. 분광 분석 (Spectroscopy)
   - 원소 조성 확인
   - 두께 측정
```

**파괴 검사** (샘플링)
```
- 접합부 단면 관찰
- 화학 성분 분석 (XRF, ICP-MS)
```

## 🔁 다중 반복 공정 (Multi-layer Fabrication)

### 단계별 반복

현대 반도체는 **10~40 단계**를 여러 번 반복하여 제조:

```
원본 웨이퍼
   ↓
[1차 반복]
산화 → 포토리소그래피 → 식각 → 세정
   ↓
[2차 반복]
증착 → 폴리싱 → 세정
   ↓
[3차 반복]
도핑 → 산화 → 포토리소그래피 → 식각 → 세정
   ↓
... (반복 계속)
   ↓
최종 웨이퍼 (수천 개 트랜지스터 포함)
```

### 왜 반복하나?
- 단층으로는 복잡한 회로 불가능
- 여러 층을 쌓아 올려야 고기능 칩 가능
- 각 층이 부품 역할

---

## ⚠️ 반도체 제조의 위험 화학약품 정리

### 극도로 위험한 물질

| 약품 | 위험성 | 취급 |
|------|--------|------|
| **HF** (불산) | 극독, 화상 위험 높음 | 전용 용기, 응급약 비치 필수 |
| **WF6** (육불화텅스텐) | 극독, 불화수소 생성 | 특수 인증 필요 |
| **SiH4** (실란) | 발화성, 유독 | 질소 퍼지 필수 |
| **AsH3** (비소화수소) | 극독, 발암성 | 폐쇄 시스템만 사용 |
| **PH3** (포스핀) | 극독, 발화성 | 격리 저장, 안전 장비 필수 |

### 환경 영향
- ✅ 폐액 처리: 중화, 응집 후 매립

## 🏢 반도체 제조와 배관 시공의 연관성

### 반도체 공장의 배관 중요성

**1. 용수(Ultra Pure Water) 공급 배관**
- DI수 대량 필요 (하루 수백만 리터)
- 초고순도 유지 필수 (1ppt 수준)
- 재질: SUS316 (부식 방지)
- 필터, 온도 제어 필요

**2. 특수 가스 배관**
- 독성/발화성 가스 운송 (실란, 비소화수소 등)
- 이중 배관, 누수 감지 시스템 필수
- 재질: SUS304, 구리(동)
- 압력, 유량 정밀 제어

**3. 폐액 처리 배관**
- 불산, 황산 등 부식성 액체 배출
- 재질: PTFE(테플론), 특수 플라스틱
- 다층 격리 시스템

**4. 냉각수 배관**
- 설비 냉각용 순환 시스템
- 온도 정밀 제어 (±0.5°C)
- 재질: 구리, SUS304

**5. 진공 시스템 배관**
- 포토리소그래피, 건식 에칭 설비용
- 극미세한 누수도 불량 원인
- 재질: 스테인리스, 특수 금속

## 💡 핵심 개념 정리

### "반도체는 화학 공장이다"
- 극미량의 불순물도 불량 원인
- 매 공정마다 정밀한 화학 반응 필요
- 온도, 압력, 시간, 농도 모두 중요

### "배관은 반도체 공장의 핵심 인프라"
- 순수 물질 운송 (용수, 가스)
- 정밀한 온도/압력 제어
- 누수 방지 및 안전
- 반도체 품질을 직접 좌우

### "미래 반도체는 더 어려워진다"
- 미세공정화 (7nm → 3nm → 2nm...)
- 화학약품 독성 증가
- 배관 정밀도 요구 수준 상승
- 환경 기준 강화
