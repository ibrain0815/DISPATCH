## 🎯 핵심 정보 요약

### 시장 상황
배관설계 소프트웨어는 크게 **3개의 시장 리더**가 지배하고 있으며, **BIM 기반 솔루션**으로 Revit이 새로운 옵션으로 부상하고 있습니다:

| 프로그램 | 개발사 | 시장지위 | 가격대 | 분류 |
|---------|--------|--------|--------|-----|
| **AVEVA E3D** | AVEVA Group | 1위 | 매우 높음 | 배관설계 전문 |
| **SmartPlant 3D** | Hexagon | 1~2위 | 매우 높음 | 배관설계 전문 |
| **CADWorx** | Hexagon | 3~4위 | 중상 | 배관설계 전문 |
| **AutoCAD Plant 3D** | Autodesk | 4~5위 | 중하 | 배관설계 전문 |
| **Revit** | Autodesk | 신규 | 중하 | BIM 통합 설계 |

> **Revit은 배관설계 전문 소프트웨어가 아닙니다.** BIM(건물정보모델) 기반의 통합 설계 소프트웨어로, 배관 설계뿐만 아니라 건축, 구조, 기계, 전기 등 모든 분야의 설계를 통합으로 수행할 수 있습니다. 따라서 소규모 프로젝트나 BIM 워크플로우가 필요한 경우 고려 대상이 될 수 있습니다.

## 💡 비전공자를 위한 설명

### 배관설계는 왜 어렵나요?

1. **복잡성**: 수천 개의 파이프, 밸브, 지지대가 3차원 공간에서 얽혀 있음
2. **정밀도**: 직경 1cm의 차이가 설치 실패를 초래할 수 있음
3. **협업**: 전기, 기계, 토목 등 여러 분야가 동시에 설계
4. **비용**: 설계 오류 = 건설 비용 증가, 공기 지연

### 좋은 배관설계 소프트웨어가 해주는 일

✅ **3D 모델링**: 복잡한 배관을 입체적으로 설계
✅ **간섭 감지**: 다른 배관/장비와 충돌하는지 자동 확인
✅ **도면 자동 생성**: 3D 모델에서 자동으로 2D 도면 추출
✅ **자재 명세서**: 필요한 파이프, 밸브의 종류와 수량 자동 계산
✅ **규칙 자동화**: 회사 표준을 설정해서 자동으로 적용

## 🔍 주요 특징 비교

### 학습 난이도
```
쉬움 ←→ 어려움

AutoCAD Plant 3D (쉬움)
    ↓
Revit (중하)
    ↓
CADWorx (중하)
    ↓
SmartPlant 3D (어려움)
    ↓
AVEVA E3D (매우 어려움)
```

**이유**: AVEVA와 SmartPlant는 배관설계 전용 기능이 많지만, Revit과 AutoCAD Plant 3D는 직관적인 BIM 인터페이스를 제공합니다. 단, Revit은 배관설계만이 아닌 건축 전체를 다루므로 초기 학습이 필요합니다.

### 가격대
```
저가 ←→ 고가

AutoCAD Plant 3D (~$2,310/년, 약 ₩3,465,000/년)
    ↓
Revit (~$680/월 또는 ~$6,500/년, 약 ₩1,020,000/월 또는 ₩9,750,000/년)
    ↓
CADWorx (~$15,000+/년, 약 ₩22,500,000+/년)
    ↓
SmartPlant 3D (견적)
    ↓
AVEVA E3D (견적, 일반적으로 가장 비쌈)
```

**참고**: Revit은 구독 기반 모델로, 단기 도입에는 저렴하지만 장기 사용 시 총비용이 증가합니다.

### 시장점유율 (배관설계 전문 소프트웨어)
```
AVEVA E3D + 이전 PDMS:  40~45%
SmartPlant 3D:          30~35%
CADWorx:                15~20%
AutoCAD Plant 3D:       5~10%
기타 (Revit, Bentley 등): 5~10%
```

**주의**: Revit은 배관설계만을 위한 소프트웨어가 아니므로, BIM 시장에서의 점유율은 매우 높지만(40% 이상), 배관설계 전문 시장에서는 신규 진입자입니다.

|------|--------------------------|------|

| **소규모/초기설계** | Revit, AutoCAD Plant 3D | 빠른 개념 설계, 비용 효율 |

## 🚀 빠른 시작

### 각 프로그램별 추천 사용자

**AVEVA E3D 추천:**
- 대형 EPC 업체 (Engineering, Procurement, Construction)
- 석유화학, 해양 플랫폼, 발전소 프로젝트
- 높은 정밀도와 클래시 감지가 중요한 경우
- 예산이 충분한 조직

**SmartPlant 3D 추천:**
- 초대형 프로젝트 (수십억 달러 규모)
- 다학제 협업 (전기, 기계, 토목 동시 설계)
- 장기적인 자산 관리까지 필요한 경우

**CADWorx 추천:**
- AutoCAD를 이미 사용 중인 조직
- 중소 규모 EPC 업체
- 비용과 기능의 균형이 중요한 경우

**AutoCAD Plant 3D 추천:**
- 작은 프로젝트 또는 부서 단위 설계
- 배관설계가 주업무가 아닌 경우
- 빠른 도입과 낮은 비용이 중요한 경우

**Revit 추천:**
- BIM 기반 통합 설계가 필요한 건축 프로젝트
- 건축, 구조, 기계, 전기, 배관이 함께 설계되는 경우
- 초기 개념 설계와 BIM 협업이 중요한 경우
- Revit 플러그인(MEP 도구)을 활용하고자 하는 경우
