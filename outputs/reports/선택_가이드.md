# 배관설계 소프트웨어 선택 가이드

## 📋 개요

이 가이드는 **어떤 프로그램을 선택해야 하는지**를 단계별로 결정하도록 도와줍니다.

> 🎯 **핵심**: 프로그램은 회사의 프로젝트 규모, 예산, 팀 역량, 전략에 따라 선택해야 합니다. "최고의 프로그램"은 없으며, "가장 적합한 프로그램"이 있을 뿐입니다.

---

## 🔍 의사결정 절차

### 1단계: 프로젝트 규모 파악

#### 질문: "우리가 주로 설계하는 프로젝트의 규모는?"

**프로젝트 규모별 분류**

```
┌─────────────────────────────────────────────┐
│  초소규모: $1M 미만                         │
│  └→ 간단한 배관 설계                        │
│     권장: 수동 설계도 또는 Plant 3D         │
│     대안: Revit (BIM 필요시)                │
├─────────────────────────────────────────────┤
│  소규모: $1M ~ $50M                         │
│  └→ 중소 산업 프로젝트                      │
│     권장: CADWorx 또는 Plant 3D             │
│     대안: Revit (건축 포함시)               │
├─────────────────────────────────────────────┤
│  중규모: $50M ~ $500M                       │
│  └→ 일반적인 EPC 프로젝트                   │
│     권장: CADWorx 또는 AVEVA E3D            │
│     건축 포함: Revit + 플러그인             │
├─────────────────────────────────────────────┤
│  대규모: $500M ~ $5B                        │
│  └→ 복잡한 산업 시설                        │
│     권장: AVEVA E3D 또는 SmartPlant 3D      │
│     주의: Revit은 기능 부족                 │
├─────────────────────────────────────────────┤
│  초대규모: $5B 이상                         │
│  └→ 석유화학, 해양, 조선                    │
│     권장: AVEVA E3D 또는 SmartPlant 3D      │
│     주의: Revit은 사용 불가                 │
└─────────────────────────────────────────────┘
```

**당신의 프로젝트 규모는?** → 위에서 찾기

### 2단계: 팀의 현재 상태 파악

#### 질문 1: "팀이 이미 사용 중인 소프트웨어?"

**시나리오별 선택**

```
❌ "AutoCAD를 많이 사용 중입니다"
  → CADWorx ★★★★★
  → AutoCAD Plant 3D ★★★★☆
  이유: AutoCAD 인터페이스 친숙, 빠른 도입

✅ "Revit을 사용 중입니다"
  → Revit MEP 모듈 ★★★★☆
  이유: BIM 통합 설계, 건축과 배관 동시 설계

✅ "특별히 사용 중인 배관설계 소프트웨어는 없습니다"
  → 프로젝트 규모와 BIM 필요 여부에 따라 선택 (아래 참고)

⚠️ "이미 PDMS/E3D를 사용 중입니다"
  → E3D 계속 사용
  이유: 마이그레이션 비용 > 신규 도입 비용

⚠️ "이미 SmartPlant 3D를 사용 중입니다"
  → SmartPlant 3D 계속 사용
```

#### 질문 2: "팀의 배관설계 경험 수준?"

```
초급 수준 (배관설계 초보자)
├─ AutoCAD 경험 있음
│  ├→ CADWorx 또는 Plant 3D (권장)
│  │   학습기간: 1~3개월
│  └→ Revit + MEP (BIM 필요시)
│      학습기간: 2~3개월
└─ AutoCAD 미경험
   ├→ Plant 3D (권장)
   │   학습기간: 2~4주
   └→ Revit (BIM 중심)
       학습기간: 1~2개월

중급 수준 (1~3년 경험)
├─ AutoCAD 기반 사용자
│  ├→ CADWorx (권장)
│  │   학습기간: 1~2개월
│  └→ Revit (업그레이드 가능)
│      학습기간: 1개월
└─ PDMS/E3D 기초
   └→ E3D (권장)
      학습기간: 3~6개월

고급 수준 (3년 이상)
├─ PDMS/E3D 숙련
│  └→ E3D (권장)
│     추가학습: 1~2개월
└─ SmartPlant 3D 숙련
   └→ SmartPlant 3D (권장)
```

### 3단계: 예산 파악

#### 질문: "3년 프로젝트에 배치할 수 있는 예산?"

**예산별 선택**

```
매우 제한적 ($10K~50K, 약 ₩15M~75M)
├→ AutoCAD Plant 3D ★★★★★ (추천)
│  전체 비용: $40~50K (약 ₩60M~75M)
└→ Revit MEP ★★★★☆ (BIM 필요시)
   전체 비용: $42~75K (약 ₩63M~112.5M)

중간 ($100K~200K, 약 ₩150M~300M)
├→ CADWorx ★★★★☆
│  전체 비용: $115~170K (약 ₩172.5M~255M)
├→ AutoCAD Plant 3D + 추가 도구
│  전체 비용: $50~80K (약 ₩75M~120M)
└→ Revit + MEP 플러그인 ★★★★☆
   전체 비용: $50~120K (약 ₩75M~180M) (BIM 프로젝트)

충분한 ($300K 이상, 약 ₩450M 이상)
├→ AVEVA E3D ★★★★★
│  전체 비용: $400~900K (약 ₩600M~1,350M)
├→ SmartPlant 3D ★★★★★
│  전체 비용: $350~850K (약 ₩525M~1,275M)
└→ Revit + 고급 MEP 환경 구축
   전체 비용: $100~300K (약 ₩150M~450M) (대규모 BIM)
```

**주의**: 라이선스 비용만이 아니라 **교육, 구현, 유지보수까지** 포함해서 생각해야 합니다.

```
│  └─ 이유: 복잡도 높음, 정밀도 필수, EPC 표준

├─ 조선 (Shipbuilding)
│  ├─ 주요: SmartPlant 3D ★★★★★
│  ├─ 차선: AVEVA E3D ★★★★☆
│  └─ 이유: 초대규모, 멀티디스플린, 통합 환경
│
├─ 발전 (Power Generation)
│  ├─ 주요: AVEVA E3D ★★★★★
│  ├─ 중간: SmartPlant 3D ★★★★☆
│  ├─ 차선: CADWorx ★★★☆☆
│  └─ 이유: 규모 클수록 E3D, 소규모는 CADWorx
│
├─ 일반 산업 (General Industry)
│  ├─ 주요: CADWorx ★★★★★

│  └─ 이유: 비용 효율, 배우기 쉬움
│
├─ 제약/식품 (Pharma/Food)
│  ├─ 주요: CADWorx ★★★★☆
│  ├─ 중간: Plant 3D ★★★☆☆
│  └─ 이유: 복잡도 중간, 위생기준 준수
│
├─ 건축 (Architecture & Engineering)
│  ├─ 주요: Revit MEP ★★★★★
│  ├─ 중간: CADWorx ★★★★☆
│  └─ 이유: BIM 협업, 다학제 통합 설계
│
├─ 건축 MEP (HVAC, 배관, 전기)
│  ├─ 주요: Revit MEP ★★★★★
│  ├─ 중간: AutoCAD Plant 3D ★★★★☆
│  └─ 이유: 건축과 동시 설계, 간섭 감지
│
└─ 교육/연구 (Academia)
   ├─ 주요: Revit ★★★★★ (BIM 교육)
   ├─ 차선: Plant 3D ★★★★★
   └─ 이유: 가장 저렴, 배우기 쉬움, 무료 라이선스
```

## ✅ 의사결정 체크리스트

다음 체크리스트를 완성해서 당신에게 맞는 프로그램을 찾으세요.

### A. 프로젝트 규모

```
□ 초소규모 ($1M 미만)                    → Plant 3D 또는 Revit 추천
□ 소규모 ($1M ~ $50M)                   → CADWorx, Plant 3D 또는 Revit 추천
□ 중규모 ($50M ~ $500M)                 → CADWorx, E3D 또는 Revit+플러그인 추천
□ 대규모 ($500M ~ $5B)                  → E3D 또는 SmartPlant 3D 추천
□ 초대규모 ($5B 이상)                   → E3D 또는 SmartPlant 3D 필수
```

### B. 현재 사용 중인 소프트웨어

```
□ Revit 사용 중                          → Revit MEP 모듈 추천
□ AutoCAD 사용 중                        → CADWorx 추천
□ 배관설계 소프트웨어 없음               → 규모와 BIM 필요 여부에 따라 선택
□ PDMS/E3D 사용 중                       → E3D 계속 사용
□ SmartPlant 3D 사용 중                  → SmartPlant 3D 계속 사용
```

### C. 팀의 경험

```
□ 배관설계 초보자 + AutoCAD 경험         → CADWorx 또는 Revit
□ 배관설계 초보자 + AutoCAD 미경험      → Plant 3D 또는 Revit
□ 배관설계 초보자 + BIM 중심             → Revit 추천
□ 배관설계 중급자                        → CADWorx, Revit 또는 E3D
□ 배관설계 고급자 (E3D 경험)             → E3D
□ 배관설계 고급자 (SmartPlant 경험)      → SmartPlant 3D
```

### D. 예산

```
□ $50K 미만 (약 ₩75M 미만)               → Plant 3D 또는 Revit
□ $50K ~ $200K (약 ₩75M~300M)           → CADWorx 또는 Revit+플러그인
□ $200K 이상 (약 ₩300M 이상)            → E3D, SmartPlant 3D 또는 Revit+고급
```

### E. 산업 분야

```
□ 석유화학 (O&G)                         → E3D 필수
□ 조선                                   → SmartPlant 3D 필수
□ 발전소                                 → E3D 권장
□ 일반 산업                              → CADWorx 추천
□ 제약/식품                              → CADWorx 권장
□ 건축/건설 (BIM 중심)                   → Revit MEP 필수
□ 건축 MEP                               → Revit 추천
□ 교육 기관                              → Plant 3D 또는 Revit 추천
```

## 🚀 단계별 구현 로드맵

선택한 프로그램을 도입할 때의 타이ム라인입니다.

### Revit MEP 도입 (총 2~4개월)

```
Month 1
├─ Week 1: 라이선스 구매 및 설치
├─ Week 2: BIM 기본 개념 및 Revit 인터페이스 교육
├─ Week 3: MEP 모듈 (배관, HVAC) 기본 교육
└─ Week 4: 간단한 테스트 프로젝트 (건축+배관)

Month 2
├─ 실제 프로젝트 적용 (초기 단계)
├─ 간섭 감지 및 BIM 협업 워크플로우 설정
├─ Revit 360 클라우드 협업 환경 구축
└─ 팀 협업 표준 정의

Month 3~4
├─ 실제 프로젝트 진행
├─ MEP 플러그인 도입 (필요시)
├─ 고급 기능 학습
└─ 최적화 및 성과 평가

결과: 3~4개월 후 정상 운영 가능 (Plant 3D와 유사)
```

---

```
Month 1
├─ Week 1: 라이선스 구매 및 설치
├─ Week 2: 기본 교육 (자체 또는 외부)
├─ Week 3: 작은 테스트 프로젝트 시작

Month 3
├─ 정상 운영
├─ 문제 해결 및 최적화
└─ 추가 기능 학습

결과: 3개월 후 정상 운영 가능
```

```
Month 1
├─ 요구사항 분석

└─ 테스트 프로젝트 시작

Month 3~4
├─ 실제 프로젝트 적용
├─ 팀 전체 교육 (1주씩)
├─ 문제 해결

Month 5~6
├─ 정상 운영
├─ 고급 기능 학습

```

### AVEVA E3D 도입 (총 12~18개월)

```
Month 1~2: 평가 및 계획
├─ 현장 실사 및 요구사항 분석
├─ 파트너사 (Systems Integrator) 선정
└─ 프로젝트 계획 수립

Month 3~4: 초기 구현
├─ 핵심 팀원 교육 (2~3주)
├─ 기본 설정 및 커스터마이징
├─ 테스트 환경 구축
└─ 파일럿 프로젝트 준비

Month 5~8: 상세 구현
├─ 파이프 스펙 복잡 데이터베이스 구축
├─ PML 매크로 개발
├─ 회사 표준 적용
├─ 상세한 문서 작성
└─ 여러 부서 협업 설정

Month 9~12: 교육 및 적용
├─ 전사 교육 (분야별 2~4주)
├─ 첫 프로젝트 적용 (지원)
├─ 문제 해결
└─ 최적화 및 미세 조정

Month 13~18: 정상화
├─ 추가 프로젝트 진행
├─ 고급 기능 활용
├─ ROI 분석
└─ 지속적 개선

결과: 18개월 후 완전 운영, 지속적 학습 필요
```

## 💡 특수 상황별 조언

### 상황 1: "빨리 도입해야 한다" (3개월 이내)

**조언**:
```
배관설계만:
→ AutoCAD Plant 3D 선택
  이유: 가장 빠른 도입 가능

BIM 통합 필요:
→ Revit MEP 선택
  이유: BIM 협업 환경 빠르게 구축, 건축과 동시 설계

대안:
→ CADWorx (AutoCAD 경험 있을 경우)

피해야 할 것:
❌ AVEVA E3D
❌ SmartPlant 3D
```

```
  이유: AutoCAD 기반, 쉬운 학습, 합리적 가격

  이유: AutoCAD 내장 기능, 가장 저렴

```

```
→ E3D 계속 사용 (무조건)

- 마이그레이션 비용 > 신규 도입 비용

- 최신 E3D 버전으로 업그레이드

```

## 🏁 의사결정 최종 정리

### "우리는 어떤 프로그램을 선택해야 할까?"

**4가지 핵심 요소**

1️⃣ **비즈니스 특성**
   - 배관설계 전문 → AVEVA E3D, SmartPlant 3D, CADWorx, Plant 3D
   - 건축(+MEP) 통합 → Revit (필수)

2️⃣ **프로젝트 규모**
   - 작다 → Plant 3D 또는 Revit
   - 중간 → CADWorx 또는 Revit+플러그인
   - 크다 → E3D 또는 SmartPlant 3D

3️⃣ **현재 기반**
   - Revit 사용 중 → Revit MEP 모듈 추천
   - AutoCAD 사용 중 → CADWorx
   - 특수 소프트웨어 없음 → 규모와 업종에 따라

4️⃣ **예산**
   - 제한적 (~$50K) → Plant 3D 또는 Revit
   - 중간 (~$150K) → CADWorx 또는 Revit+플러그인
   - 충분 ($300K+) → E3D 또는 SmartPlant 3D
