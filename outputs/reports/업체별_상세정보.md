## 1️⃣ AVEVA Group - E3D Design (PDMS 후속)

### 🏢 회사 정보

**기본 정보**
- 회사명: AVEVA Group plc
- 본사: 영국 런던
- 설립: 1998년 (현재 형태의 AVEVA)
- 역사: PDMS는 1980년대부터 시작 → 2007년 AVEVA로 통합 → 현재 E3D로 진화

**회사 규모**
- 직원 수: 약 4,000명 (전 세계)
- 시장: 50개 이상 국가
- 주요 비즈니스: 산업 소프트웨어, 디지털 트윈

**모회사 변화**
- 2007: PDMS 개발사 E.ON → AVEVA로 인수
- 2020: Schneider Electric 자회사 → 독립 (Axiom Software로 스핀오프)
- 2021: Aveva.com으로 통합 및 현재 형태

### 📦 E3D Design 제품

**제품명**: AVEVA E3D Design

**역사**
```
1980년대  → PDMS 개발 (가장 오래된 배관설계 소프트웨어)
2010년대  → E3D로 진화 시작
2024년    → PDMS 단계적 단종 공지
2025년~   → E3D Design 완전 전환
```

**주요 기능**

| 기능 | 설명 |
|------|------|
| **3D 모델링** | 배관, 장비, 구조물, HVAC의 정밀 3D 설계 |
| **간섭 감지** | 자동 clash detection (업계 최고 수준) |
| **자동 도면화** | 3D→ 2D 아이소메트릭 자동 생성 |
| **자재 관리** | 복잡한 BOM 자동 생성 |
| **규칙 기반** | PML 스크립트로 무한 자동화 |
| **협업** | 실시간 멀티유저 편집 지원 |
| **통합** | 구조, 전기, 계측 모두 포함 |

**포함되는 모듈**
- **MODEL**: 3D 모델링 및 데이터 관리
- **DRAW**: 2D 도면 생성
- **ISODRAFT**: 아이소메트릭 도면 자동화
- **구조/강재**: 구조물 및 강재 설계
- **HVAC**: 환기 및 냉난방 시스템

### 📦 Revit MEP (기계, 전기, 배관)

**제품명**: Autodesk Revit

**특징**
- BIM(Building Information Model) 기반 통합 설계 소프트웨어
- MEP 모듈: 배관, 덕트(냉난방), 전기 케이블 통합 설계
- 배관설계 전문 소프트웨어가 아님 (건축 전체 설계 플랫폼)
- 건축, 구조, 기계, 전기, 배관이 단일 모델에서 협업

**주요 기능**

| 기능 | 설명 |
|------|------|
| **MEP 통합** | 배관, 덕트, 케이블 트레이 한번에 설계 |
| **간섭 감지** | 각 분야 간 간섭 자동 감지 |
| **자동 라우팅** | 배관 경로 자동 최적화 (플러그인) |
| **BIM 360** | 클라우드 기반 협업 플랫폼 |
| **연동 설계** | 건축과 동시에 배관 설계 |
| **3D 시뮬레이션** | 시공성 검토 (4D 애니메이션) |
| **도면 자동화** | 3D에서 자동으로 평면도, 단면도 추출 |
