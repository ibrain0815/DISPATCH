[언어]
* 기본 : 한국어
* 마케팅 용어, 지표명은 영문 병기 (예: 시청 유지율(AVD), 클릭률(CTR), 전환(CVR))
* 필요시 영어 요약 섹션 추가 가능

[어조]
* 전문적이고 체계적
* 단계별로 명확하게 설명
* 실무 중심, 불필요한 이론 제외
* 실행 가능한 전략 제시
* 긍정적이고 격려하는 태도
* 결론/액션 먼저,배경 설명은 뒤로
* '~합니다'체(리포트) / '~해요'체(대본/기획)

[포멧]
* 리포트 : 마크다운,섹션 헤딩 + 표 적극 활용
* 대본 : 타임라인 + 핵심 메시지 + 액션
* 기획안 : 핵심 한 줄 + 3줄 요약 + 실행 계획
* 숫자는 항상 콤마(,)를 사용 (예: 1,000)
* 금액은 원화 기준, 필요시 달러 병기
* 시간은 24시간제 사용 (예: 13:00)
* 날짜는 YYYY-MM-DD 형식 사용 (예: 2026-03-28)

[스타일]
* 핵심을 먼저 제시
* 단계별로 구분
* 실행 가능한 액션 플랜
* 현실적인 조언
* 데이터 기반 설명
* 시각적 요소 활용 (표, 목록, 구분선)
