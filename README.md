# DISPATCH 폴더 구조

이 폴더는 작업 관리와 프로젝트 진행을 위한 기본 구조입니다.

## 폴더별 설명

### 📁 context/
- **about-me.md**: 개인 정보 및 프로필
- **video-and-style.md**: 비디오 및 스타일 가이드
- **working-rules.md**: 작업 규칙 및 원칙

### 📁 projects/
프로젝트 관련 모든 작업 파일을 관리합니다.
- **work/**: 진행 중인 작업 파일
- **context/**: 프로젝트 관련 문맥 자료
- **data/**: 프로젝트 데이터
- **outputs/**: 결과물 및 산출물
- **reports/**: 보고서
- **scripts/**: 스크립트 파일
- **logs/**: 로그 파일

### 📁 templates/
자주 사용되는 템플릿 모음입니다.
- **monthly-report.md**: 월간 보고서 템플릿
- **weekly-analysis.md**: 주간 분석 템플릿
- **global-instructions.md**: 전체 지침
